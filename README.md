## Scripts
Below you will find some information on how to run scripts.

#### `npm run compile`
- Compile and watch Jekyll, Sass and Javascript.


#### `npm run browsersync`
- Setup auto refresh browser when html and css changes. Depends on Jekyll server to run.


#### `npm start`
